Para ejecutar el sistema es necesario correr el backend y el frontend.
Utilizas dos terminales distintas y dentro del proyecto se colocan los siguientes comandos:

1. cd backend
2. npm run dev
3. cd frontend
4. npm start

El backend se ejecuta en http://localhost:5000 y el frontend en http://localhost:3000

Para ejecutar las pruebas es necesario correr el siguiente comando:
1. cd frontend
2. npm run cy:open
